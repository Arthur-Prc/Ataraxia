# Ataraxia

**Ataraxia** is an open, recurring **household subsistence and resilience protocol** focused on predictable resource production and **non-speculative, hardware-backed income generation**.

This project documents how an ordinary house or apartment can sustainably produce **food, energy, compute, and digital value**, combining physical subsistence with **Web3 infrastructure roles** such as mining and node operation â€” not bandwidth resale, not hype-driven token schemes.

Ataraxia is designed to be:

* Calm
* Boring
* Predictable
* Auditable

---

## Design Principles

* **Small-space compatible** (apartment or small house)
* **Low-to-moderate startup cost**
* **Hardware-backed value** (real resources, not promises)
* **Composable modules** (each unit stands alone)
* **Automation-first mindset**
* **Energy-aware** (electricity and heat are first-class constraints)
* **Legality-first** (no illegal production or evasion)

## Modular Repository Structure

Each production line is a **self-contained module**:

* `food/mushrooms`

* `protein/quail`
